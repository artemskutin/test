# Инструкция по Markdown

Что бы добавить ненумерованные списки необходими пункты выделить (*) например:
* Элемент 1
* Элемент 2
* Элемент 3

Что бы добавить номированые списки нужно просто их пронумеровать, например вот так:
1. Элемент первый
2. эдемент второй 
3. Элемент третий
